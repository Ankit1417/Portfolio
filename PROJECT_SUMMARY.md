# Portfolio Website - Project Summary

## 📋 Overview

A modern, professional portfolio website for **Ankit Gupta**, showcasing expertise in Data Analysis and Full Stack MERN Development. Built with React, Vite, Tailwind CSS, and Framer Motion.

---

## ✅ What's Included

### 🎨 Complete Website Sections

1. **Navbar** - Responsive navigation with smooth scroll
2. **Hero Section** - Eye-catching introduction with CTAs
3. **About Me** - Professional background and highlights
4. **Skills** - Categorized technical expertise display
5. **Projects** - Featured work with detailed descriptions
6. **Education** - Academic background
7. **Contact** - Contact form and social links
8. **Footer** - Professional footer with branding

### 🛠️ Technical Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Smooth scroll animations with Framer Motion
- ✅ Modern UI with Tailwind CSS
- ✅ Professional color scheme (Indigo & Cyan)
- ✅ Hover effects and transitions
- ✅ SEO-optimized structure
- ✅ Fast loading with Vite
- ✅ Clean, maintainable code
- ✅ Production-ready build system

## 🎯 Design Specifications

### Color Palette
- **Primary**: #4F46E5 (Indigo Blue)
- **Secondary**: #06B6D4 (Cyan)
- **Accent**: #0F172A (Navy/Deep Slate)
- **Background**: #F9FAFB (Light Gray-White)

### Typography
- **Font Family**: Inter (Google Fonts)
- **Headings**: Bold, 2xl-5xl sizes
- **Body**: Regular, 16-18px

### Responsive Breakpoints
- **Mobile**: < 768px
- **Tablet**: 768px - 1023px
- **Desktop**: ≥ 1024px
